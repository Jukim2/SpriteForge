? What is Rematch?
It makes all exported sprites the exact same size, which is required by game engines like Unity or Unreal for sprite animations.

Size Mode:
  • Auto (Largest Cell): Finds the biggest sprite you sliced, and makes every other sprite that same size.
  • Custom Size: You manually set the exact Width/Height for all sprites.
Fit Mode:
  • Contain: Places the sprite in the center and adds transparent padding around it so it isn't stretched.
  • Cover: Crops the outer edges of the sprite if it is larger than the target size.
  • Stretch: Forcibly resizes the sprite to fill the target size, which may stretch/squash it.
